4 Queens Puzzle on a 4x4 Chess Board

Personal Note

This was originally my lab assignment for a Data Structures class, where I implemented the N-Queens problem using backtracking. I got bored of printing 1s and 0s to the console, so I decided to play with it using HTML, CSS, and JavaScript to create this visualizer.

Instead of raw arrays, you can now see queens placed on the chessboard and cycle through all valid solutions.
